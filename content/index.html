This is a community-driven open source project to make working with TTCN-3 more productive.

Get in touch

  • TTCN-3 Standardization Home: The official TTCN-3 homepage provides the standard documents, tutorials and much more.

  • TTCN-3 Mailing List: The official mailing list, for general questions and discussions.

  • Change Requests: The official tracker for change requests, bug reports, ... for the TTCN-3 standard.
    Kindly read the howto for more details.

  • Reddit: The TTCN-3 subreddit, for all kinds of questions, discussions and announcements.

  • Stack Overflow: Developers and testers around the world helping each other out.

  • TTCN-3 Professionals (LinkedIn): The TTCN-3 Professionals group on LinkedIn provides a platform for new ideas, job offers, meetups and more.

Resources

  • Standards: The official TTCN-3 standard documents.

  • Cheat Sheet (PDF): A quick reference card for TTCN-3 provides you a good overview of the language syntax.

  • Refactoring Catalog: This catalog provides a list of refactoring operations for TTCN-3 source code.

  • Tutorials: A collection of various tutorials for TTCN-3 curated by ETSI.

  • Repository for TTCN-3 Conformance Tests: This working repository contains the sources for the TTCN-3 conformance tests.

  • Repository for TTCN-3 Standard: This working repository provides the files needed to create the TTCN-3 standard.

  • Mantis Archive: This folder contains the archived change requests for the TTCN-3 standard.

  • Meeting Minutes: This folder contains the summaries of the TTCN-3 work sessions.

Test Suites, Examples and Libraries

  • TTCN-3 Conformance tests: Official TTCN-3 conformance tests.

  • 3GPP tests: UE test suites for 5G, LTE, IMS, UTRA, Positioning and MCS.

  • ETSI tests: Various test suites. IPv6, SIP, WiMax and more.

  • oneM2M test suites: Service layer test suite.

  • Osmocom Cellular Network tests (mirror): Osmocom cellular network tests.

  • Intel Net test suites: TCP protocol test suite.

  • ETSI Examples: TTCN-3 examples provided by ETSI.

  • ETSI Libraries: Libraries with common code, messages and types.
