## Overview

ntt is a tool-set, which helps you working with TTCN-3. TTCN-3 is a simplified
programming language, mostly used for writing integration or protocol tests. If
you want to learn more about TTCN-3, [this page]({{ "/ttcn3" | relative_url }})
should have you covered.

ntt provides a language server for TTCN-3. This language server adds
intelligent language features (auto completion, go to definition, ...) to your
favourite editor.  [Check out the editor page]({{ "/editors" | relative_url }})
for a full feature-list and instructions for installation and configuration.
