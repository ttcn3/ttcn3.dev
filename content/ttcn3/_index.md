# What is TTCN-3

[TTCN-3](http://ttcn-3.org) is a standardized
domain-specific language for scripting tests. It supports
various types of testing and is a good addition to unit tests.
Despite being clearly from the last millennium, TTCN-3 offers
some excellent features for testing protocols and services:


* native support for concurrency and alternate code flows (similar to Go).
* a powerful matching mechanism. Think supercharged regular expressions for arbitrary data types.
* and it’s language agnostic: Adapters and codecs are implemented in any language you see fit. So you can leverage any ecosystem you need.
o
, like integration tests for example.<br>
			The idea behind TTCN-3 is to offer developers a
			familiar environment, without the hassle of general
			purpose language.
