# Modern tools for TTCN-3

ntt is a free and open tool-set for testing with TTCN-3. It gives you the tools
you need to write tests efficiently. Get started and find out how ntt can help
you.  <a href="/docs/getting-started" role="button">Get started</a>


## What is TTCN-3?

TTCN-3 is a programming language specialized for writing tests that
exceed simple unit testing. It has built-in support for concurrency,
network communication and pattern matching. TTCN-3 is also an ETSI
standard and well known in industry and telecommunication.
[Learn more](/ttcn3)

## Universal IDE support <!--class="fa fa-pencil" aria-hidden="true" -->
Use your favourite editor and unlock advanced IDE features.  ntt is
a universal TTCN-3 language plugin and provides IntelliSense,
code navigation, find all references and [more.](/editors)

## Developer friendly</h2></a> <!-- class="fa fa-users" aria-hidden="true" -->
Focus on writing tests. Your time is precious, therefore ntt gets out of
your way. It is easy to install, has good defaults and provides a package
manager for TTCN-3, too.

## Powerful CLI <!-- class="fa fa-cogs" aria-hidden="true" -->
Automate everything! ntt is designed to support power users and
large scale environments alike. Custom templates and machine readable
output help embedding ntt into your workflows as well as Bash and
Python scripts
